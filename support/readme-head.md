CSS utils
=========

Collection of Sass utilities.

How to use
----------

Each stylesheet is self-contained, so simply pick out what you need and put it 
in your project.

[View stylesheets >]( sass/ )

#### Example

~~~ sh
cd ~/my/project/sass/
wget https://raw.github.com/rstacruz/cssutils/master/sass/_antialias.sass
~~~
